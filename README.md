# My Tasks — To-Do List App

A clean, modern to-do list app built with pure HTML, CSS, and JavaScript. No frameworks, no libraries — just vanilla web development.

![My Tasks App](screenshot.png)

---

## Live Demo

> Open `index.html` in your browser — works instantly, no setup needed.

---

## Features

- Add new tasks
- Mark tasks as complete
- Delete tasks
- Filter by All / Active / Completed
- Live stats — Total, Active, Done
- Clear all completed tasks at once
- Data saved in localStorage — tasks stay after refresh
- Responsive design
- Press Enter to add a task

---

## Built With

- HTML5
- CSS3
- JavaScript (Vanilla)
- Google Fonts — Syne + DM Sans
- localStorage API
